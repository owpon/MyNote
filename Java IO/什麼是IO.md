[[什麼是IO]]

I / O  -> 簡單的解釋就是電腦的輸入跟輸出

以一般文件為例，需要從硬碟讀到用戶空間(我推測應該是類似一個緩衝區的意思)，這樣的操作就可以稱做一次I/O的操作，更準確的說是一次文件I/O的操作

我們瀏覽網頁的時候，發出一個request給serve，然後server通過網路回傳資料給我們時，此時程序將數據從TCP緩衝區複製到用戶空間，這樣也是一個I/O，更準確的說是網路I/O
